## 项目的技术栈

1. Node.js
2. MongoDB
3. JavaScript



## 功能实现

#### 用户管理

- 创建用户 （可以设置管理员权限）
- 用户登录 （判断权限 登录前台首页or后台管理页面）
- 退出登录
- 操作用户（增删改查）



#### 分类管理

- 添加分类 （文章的分类，作为前台样式）
- 查询分类



#### 文章管理

- 创建文章（绑定登录用户的id）
- 查询文章
- 根据登录的id修改文章和删除文章
- 添加文章点赞和搜索功能



#### 评论管理

- 添加评论（绑定文章的id）
- 后台更改评论状态
- 操作评论



#### 网站设置

- 添加轮播图功能
- 获取网站配置
